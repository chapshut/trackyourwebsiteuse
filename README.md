# Time Tracker — Private & Beautiful

A minimal, fast, 100% local time tracker that shows:
- Today’s time per website (live)
- Daily average (all time)
- Weekly average (last 7 days)
- Full domain names (no cutoff)
- One-click wipe all data

**No data ever leaves your browser. No tracking. No accounts.**

---

### How to Install & Use (Brave, Chrome, Edge, etc.)

1. **Download or copy** this entire folder somewhere safe on your computer  
   (e.g. `~/TimeTracker/` or `C:\TimeTracker\`)

2. Open your browser and go to the extensions page:
   - Brave / Chrome / Edge: `brave://extensions/` or `chrome://extensions/`

3. Enable **Developer mode** (toggle in the top-right corner)

4. Click **"Load unpacked"**

5. Select this folder (the one containing `manifest.json`)

6. The extension icon (clock) will appear in your toolbar

7. Click the icon anytime to see your live stats

Done! It starts tracking immediately.

---

### Features

- Works on all tabs (even ones open before install)
- Live updating (no need to refresh popup)
- Full website names visible
- Beautiful, readable layout
- Wipe all data with one click
- No internet permission → 100% private

---

### Uninstall or Reset

- To remove: go to `brave://extensions/` → click "Remove"
- To wipe data only: open the extension → click **"Wipe All Data"**
